# System Monitor (Qt / C++)

A real-time system monitoring application built using C++ and Qt.

## Features
- Live CPU usage tracking
- Real-time RAM monitoring
- Dynamic graph visualization
- Process listing using Windows API
- Memory usage per process
- Sorting support

## Tech Used
- C++
- Qt (Qt Widgets + Qt Charts)
- Windows API
